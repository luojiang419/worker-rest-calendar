# Changelog

## 0.1.14+14 · 精致桌面摆件套件

- 桌面摆件新增排班、时钟、记事本、专注计时四种类型，可从设置页和 Windows 原生右键菜单即时切换并持久化。
- 时钟显示秒、日期、星期、今日排班与下次休息日，按真实本地时间刷新并在跨午夜后自动校正。
- 记事本支持 4000 字多行便笺、500ms 防抖自动保存和独立拖动手柄；锁定位置后仍可正常编辑，内容进入备份导入导出。
- 专注计时器提供 25 分钟开始、暂停、继续、重置和完成状态，依据目标结束时间校正，避免系统睡眠或界面阻塞造成计时漂移。
- 新增摆件共享既有设计令牌、透明安全边距和无描边卡片，覆盖五种视觉风格、浅色/暗黑模式、小中大尺寸及 130% 字体。
- 数据库升级到 v8；格式化、静态检查与 231 项单元/Widget 测试通过，Windows 集成测试和安装包由云端发布工作流验收。

## 0.1.13+13 · 可量化安装进度

- 独立更新器将 Inno Setup 从完全不可见的 `/VERYSILENT` 调整为 `/SILENT`，显示标准 Windows 安装进度窗口。
- 安装窗口明确展示目标版本、当前安装动作、正在处理的文件、真实进度条和 0–100% 百分比。
- 百分比直接来自 Inno Setup 的 `CurInstallProgressChanged`，不再使用循环动画或按时间伪造进度。
- 安装期间继续禁止取消，避免半安装状态；非零退出码仍由独立更新器显示失败原因并保留诊断日志。
- 云端发布构建继续以 Inno Setup 6 编译脚本，语法或安装器生成失败将阻断正式 Release。

## 0.1.12+12 · 自动更新端到端验证

- 设置页新增自动更新、手动更新、禁止更新三态，以及自动检测代理、手动代理、直连三种网络模式。
- 软件启动时可从公开 GitHub Latest Release 检测新版，精确匹配 Windows x64 安装包并校验大小与 SHA-256。
- 支持“立即更新”和“下载后下次启动时更新”，由安装目录外的独立更新器关闭旧版、静默覆盖并重启新版。
- GitHub Actions 自动递增三段语义版本，完成测试、Windows 云端编译、Inno Setup 打包、草稿资产远端复核和正式 Release 发布。
- 本版本作为已安装 `0.1.11+11` 的真实升级目标，用于端到端验证检测、下载、弹窗、安装和重启链路。

## 0.1.10+11 · 首页剩余工作日计数修正

- 修复首页“再上 X 天班”误用距离下次休息日的日历天数，导致大周周二到周六显示 4 天而非 3 天。
- 现在从明天起统计至下次休息日之前的实际工作日，不计入今天和休息日，并遵循调休、请假等最终排班覆盖。
- 新增大小周中当前大周、手动覆盖和当天休息的回归测试。
- 格式化、静态检查、202 项单元/Widget 测试及 1 项 Windows 集成测试通过，并重新生成 Windows Release 与 0.1.10 安装包。

## 0.1.9+10 · 桌面摆件跨分辨率定位

- 桌面摆件位置从显示器绝对坐标升级为 V2 自适应定位：靠近边缘时保持边距，中间区域按可移动范围比例换算。
- Windows 新增分辨率、DPI 和任务栏工作区原生变化通道，显示环境稳定后自动重新定位摆件。
- 本地物理显示器与远程虚拟显示器分别恢复位置；从 1080p 切换到 2K 并返回后，不再因旧像素坐标产生漂移或越界。
- 旧 V1 绝对坐标首次读取时自动迁移，损坏数据安全回退；完整主窗口移动不再覆盖桌面摆件位置。
- 新增定位算法、V2 存储、原生通道、分辨率往返和远程显示器接入/断开回归测试；格式化、静态检查、198 项单元/Widget 测试及 1 项 Windows 集成测试通过。

## 0.1.8+9 · 跨日自动刷新

- 新增共享本地日期时钟，在下一次本地午夜自动发布新日期，不再需要重启软件。
- 应用从后台或系统睡眠恢复时立即校正日期，并重新安排下一次午夜刷新，避免暂停期间错过定时器。
- 今日首页、日历当日高亮、连续工作统计、Windows 桌面摆件和运行中的手机主屏小组件统一订阅日期变化。
- 新增午夜边界、前台恢复和今日页面跨日回归测试；格式化、静态检查、177 项单元/Widget 测试及 1 项 Windows 集成测试通过，并重新生成 Windows Release 与 0.1.8 安装包。

## 0.1.7+8 · 设置中心与自由方向月历

- 完整主窗口左侧栏左下角在主题图标下方新增设置入口，并以保留左侧栏的独立页面展示设置。
- 设置页新增开机自启、日历左右/上下滚动方向，并集中提供摆件尺寸、日期形状、今天突出方式、透明度、置顶、锁定、外观、提醒和数据入口。
- 应用数据库升级到 v7，日历滚动方向持久化并进入导入导出；旧数据库和旧备份默认使用左右滚动。
- 月历移除整页吸附和滚动回弹，可按像素停在任意位置，包括同时显示相邻两个月；视口中心月份使用正常明度，相邻月份降低明度。
- 鼠标滚轮和触控板由“滚一下切整月”改为连续像素滚动，横向与纵向共享同一惰性月份数据源。
- 从完整主窗口修改摆件尺寸、透明度、置顶或锁定时只保存摆件配置，不再改变当前主窗口尺寸和透明度。
- 格式化、静态检查、173 项单元/Widget 测试及 1 项 Windows 集成测试通过，并重新生成 Windows Release 与 0.1.7 安装包。

## 0.1.4+5 · Windows 退出性能优化

- 优化托盘“退出”时序：窗口销毁与托盘原生清理由串行改为并行，避免等待 `trayManager.destroy()` 时界面残留并表现为卡顿。
- Windows 退出时先立即隐藏窗口，再解除关闭拦截并销毁宿主，让用户点击退出后获得即时反馈。
- 退出操作增加幂等保护，连续点击或重复回调只会执行一次托盘与窗口销毁。
- 新增慢速托盘清理、原生窗口退出顺序和重复退出回归测试。
