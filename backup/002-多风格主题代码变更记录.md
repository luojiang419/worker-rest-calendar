# Backup 002 · 多风格主题代码变更记录

## 备份范围

本文档仅记录本轮主题功能的代码变更和恢复要点，不复制 `build`、`.dart_tool` 或其他可重建缓存。上一备份版本为 `001-Phase00至03代码变更记录.md`。

## 变更前

- 应用仅有 `ThemeMode.system/light/dark`。
- 桌面左侧栏按钮依据当前 `Brightness` 在浅色与暗色之间往返切换。
- `AppPreferences` 仅持久化 `themeMode`。
- `AppTokens` 仅有 `light` 与 `dark` 两份固定令牌。
- 数据库 schema 版本为 3。

## 变更后

- `lib/core/theme/app_visual_style.dart` 新增 5 种视觉风格枚举、中文名称和描述。
- `AppTokens.resolve` 依据风格与明暗生成 10 套令牌。
- `AppTokens` 新增 `visualStyle`、`backgroundAccent`、`surfaceHighlight`、`borderWidth`、`blurSigma`。
- `AppTheme.lightFor/darkFor` 将视觉风格注入 Flutter `ThemeData`。
- `app_card.dart`、`app_button.dart`、`app_bottom_sheet.dart` 根据材质令牌渲染边框、阴影、渐变和模糊。
- `theme_picker_sheet.dart` 提供明暗模式与视觉风格选择。
- `AppPreferences.visualStyle` 与数据库 `app_settings.visual_style` 完成持久化。
- 数据库 schema 版本为 4，v1/v2/v3 可升级，旧记录默认 `classic`。
- 备份 JSON 新增 `visualStyle`，缺少字段的旧备份默认恢复为 `classic`。

## 关键前后代码

### 持久化模型之前

```dart
const AppPreferences({
  this.themeMode = AppThemePreference.system,
  this.locale = 'zh_CN',
});
```

### 持久化模型之后

```dart
const AppPreferences({
  this.themeMode = AppThemePreference.system,
  this.visualStyle = AppVisualStyle.classic,
  this.locale = 'zh_CN',
});
```

### 主题构建之前

```dart
static ThemeData get light => _theme(Brightness.light, AppTokens.light);
static ThemeData get dark => _theme(Brightness.dark, AppTokens.dark);
```

### 主题构建之后

```dart
static ThemeData lightFor(AppVisualStyle style) =>
    _theme(Brightness.light, AppTokens.resolve(style, Brightness.light));
static ThemeData darkFor(AppVisualStyle style) =>
    _theme(Brightness.dark, AppTokens.resolve(style, Brightness.dark));
```

### 数据库之前

```dart
int get schemaVersion => 3;
TextColumn get themeMode => text().withDefault(const Constant('system'))();
```

### 数据库之后

```dart
int get schemaVersion => 4;
TextColumn get themeMode => text().withDefault(const Constant('system'))();
TextColumn get visualStyle => text().withDefault(const Constant('classic'))();
```
