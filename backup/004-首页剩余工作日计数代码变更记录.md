# 004 首页剩余工作日计数代码变更记录

## 变更范围

本记录只保存 0.1.9+10 到 0.1.10+11 中“再上 X 天班”计数修正的具体变化，不复制完整源码。

## 核心变更 1：分离日历日差与实际工作日

### 变更前

`TodayDashboard` 只有 `daysToNextRest`，该数值是从今天到下次休息日的日历偏移。

```dart
final daysToRest = schedule.engine.daysToNextRest(today);
```

### 变更后

保留日历偏移用于定位下次休息日，另外计算从明天到休息日前一天的最终排班工作日数。

```dart
final remainingWorkDaysBeforeNextRest = daysToRest <= 1
    ? 0
    : schedule
          .days(today.addDays(1), daysToRest - 1)
          .where((day) => day.effectiveKind.isActualWork)
          .length;
```

## 核心变更 2：倒计时卡使用正确口径

### 变更前

```dart
': 再上 $daysToNextRest 天班'
```

### 变更后

```dart
': 再上 $remainingWorkDays 天班'
```

`daysToNextRest == 0` 仍专门对应“今天就是休息日”，避免破坏原有语义。

## 核心变更 3：真实大周回归场景

测试使用大小周模式，将 2026-07-13 周一设为大周锚点：

```dart
final profile = testProfile().copyWith(
  anchorDate: CalendarDate(2026, 7, 13),
  anchorWeekType: WeekType.big,
);
```

在 2026-07-14 周二断言：

```dart
expect(dashboard.daysToNextRest, 4);
expect(dashboard.nextRestDate, CalendarDate(2026, 7, 18));
expect(dashboard.remainingWorkDaysBeforeNextRest, 3);
```

Widget 回归同时断言展示“再上 3 天班”且不展示“再上 4 天班”。
