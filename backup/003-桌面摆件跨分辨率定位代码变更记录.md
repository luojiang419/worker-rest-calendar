## 变更一：位置表达方式

### 修改前

```dart
positions[displayId] = <String, double>{
  'dx': position.dx,
  'dy': position.dy,

```

### 修改后

```dart
placements[displayId] = <String, Object?>{
  'schemaVersion': 2,
```

## 变更二：恢复方式

### 修改前

```dart
if (display != null) await windowManager.setPosition(position);
```

### 修改后

```dart
final position = placement.resolve(
  windowSize: currentWindowSize,
  workArea: currentWorkArea,
);
await windowManager.setPosition(Offset(position.x, position.y));
```

## 变更三：显示配置变化

### 修改前

Windows runner 未向 Dart 转发同一显示器的分辨率或 DPI 变化，应用只能在重新进入摆件时尝试恢复旧坐标。

### 修改后

```cpp
case WM_DISPLAYCHANGE:
  NotifyDisplayConfigurationChanged("display");
  break;
case WM_DPICHANGED:
  NotifyDisplayConfigurationChanged("dpi");
  break;
case WM_SETTINGCHANGE:
  if (wparam == SPI_SETWORKAREA) {
    NotifyDisplayConfigurationChanged("work-area");
  }
  break;
```

Dart 收到通知后等待 350ms，再根据稳定后的工作区恢复位置。

## 变更四：窗口模式隔离

### 修改前

```dart
void onWindowMoved() {
  unawaited(_savePosition());
}
```

完整主窗口拖动也可能覆盖摆件位置。

### 修改后

```dart
void onWindowMoved() {
  if (_isWidgetMode && !_applyingPosition) unawaited(_savePosition());
}
```

仅用户拖动桌面摆件时更新 V2 记录。
