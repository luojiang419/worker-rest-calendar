# Codex Master Prompt

你正在实现“工作日历”。请先阅读根目录 `AGENTS.md`，再阅读本阶段文档。严格按 Phase 顺序开发。

## 总目标

交付 iOS、Android、macOS、Windows 可运行的 Flutter 应用：支持双休、单休、大小周和自定义循环；具有今日首页、日历、单日调整、提醒、统计、浅色/暗黑主题、桌面挂件和手机主屏小组件。

## 关键领域定义

大周是周六周日双休；小周是周六上班、周日休息。任何代码、测试和文案都不得把该定义反转。

## 每次工作方式

1. 阅读当前 Phase 的目标与退出条件。
2. 先检查仓库现状，不删除已通过测试的实现。
3. 列出小步计划。
4. 实现并补充测试。
5. 运行 format、analyze、test。
6. 更新 `PROGRESS.md` 和必要文档。
7. 达到退出条件后停止，不擅自进入下一 Phase。
