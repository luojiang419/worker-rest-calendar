# 08 · 桌面挂件与手机主屏小组件

## 1. 桌面挂件

### 小型 180x220 逻辑像素

- 今日状态。
- 距离下一休息日。
- 当前大周/小周标签。

### 中型 360x220

- 今日状态。
- 本周 7 天节奏。
- 下一休息日期。

### 大型 420x360

- 月历。
- 状态图例。
- 点击日期打开完整应用对应日期。

### 桌面行为

- 无边框、圆角、透明背景。
- 可拖拽；可锁定位置。
- 可置顶。
- 透明度范围 70%–100%。
- 记住每个显示器的位置和尺寸。
- 支持开机启动。
- 右键菜单提供快捷设置。
- 主题可跟随系统，也可固定。

平台能力通过 `DesktopWindowService` 抽象，不在 Widget 内直接调用插件。

## 2. iOS/Android 主屏小组件

Flutter 负责计算并写入共享数据；平台原生界面负责渲染：

- iOS：WidgetKit/SwiftUI。
- Android：原生 AppWidget 或 Jetpack Glance。

小组件不应依赖 Flutter 引擎常驻。每次数据变化后写入紧凑快照：

```json
{
  "date": "2026-07-12",
  "todayKind": "rest",
  "weekType": "big",
  "daysToNextRest": 0,
  "week": ["work", "work", "work", "work", "work", "rest", "rest"],
  "theme": "system"
}
```

## 3. 刷新触发

- 修改排班后。
- 日期跨天后。
- 主题变化后。
- 节假日数据更新后。
- 应用恢复前台时校验快照版本。
