# 10 · 测试与验收

## 单元测试

### 排班引擎

- 双休、单休每个星期位置。
- 大小周锚点周、前一周、后一周。
- 大周周六休息，小周周六工作。
- 跨年、闰年、锚点前日期。
- 自定义循环正向与负向日期。
- 覆盖优先级。
- 下一休息日与连续工作天数。

### 数据层

- CRUD。
- 软删除。
- 导入事务回滚。
- schema 迁移。

## Widget 测试

- 浅色与暗黑主题关键组件快照。
- 字体放大时无 overflow。
- 日期状态有文字/语义，不只依赖颜色。
- 首页 loading、empty、error、normal 状态。

## 集成测试

1. 首次启动选择大小周，本周设为小周。
2. 验证本周周六显示上班、周日显示休息。
3. 修改周六为调休休息。
4. 首页和桌面/主屏小组件同步更新。
5. 启用提醒并验证通知计划。
6. 导出数据、清空、导入后恢复。

## UI 验收

- 浅色与暗黑模式均完整，无硬编码浅色背景。
- 主卡片、按钮、分段控件、弹层均有主题适配投影。
- 动画不卡顿，列表滚动无明显掉帧。
- 小屏设备与桌面窗口缩放均可用。
- Windows 桌面摆件在 1080p、2K、不同 DPI 和远程虚拟显示器往返后保持边缘距离或中间相对位置，并完整位于当前工作区。

## 发布门槛

- `flutter analyze` 无 error。
- 全部单元测试通过。
- 关键集成流程通过。
- Windows/macOS/iOS/Android 至少各完成一次 release 构建。
- 隐私说明、数据导出和清空功能可用。
