# 11 · 隐私、安全与发布

## 隐私原则

- 默认无需登录。
- 排班、请假和备注存储在本地。
- 云同步必须由用户主动开启。
- 提醒文案默认不展示敏感备注。
- 提供一键导出和一键清空。

## 安全

- 不在日志中输出备注、令牌和完整同步 payload。
- 云端令牌使用平台安全存储。
- 导入文件限制大小并校验 schema。
- 数据库迁移必须有回滚/备份策略。

## 发布资产

- App icon 与小组件图标。
- 浅色/暗黑商店截图。
- 隐私政策。
- 版本说明。
- Windows 安装包、macOS 签名与公证、iOS/Android 商店构建。

## 版本建议

- 0.1：排班引擎与内部预览。
- 0.5：完整本地 MVP。
- 0.8：桌面与主屏小组件。
- 1.0：四端发布、导入导出、稳定性达标。
