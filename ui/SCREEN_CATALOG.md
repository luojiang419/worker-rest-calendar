# UI 页面目录与状态

| 编号 | 页面 | 关键状态 |
|---|---|---|
| S01 | 欢迎页 | 首次进入 |
| S02 | 班制选择 | 未选择、已选择 |
| S03 | 大小周锚点 | 大周、小周、日期选择、预览 |
| S04 | 自定义周期 | 编辑、校验错误、预览 |
| S05 | 今日首页 | 工作、休息、调休上班、调休休息、请假 |
| S06 | 月历 | 正常、跨月、含覆盖、空数据 |
| S07 | 周历 | 大周、小周、自定义周期 |
| S08 | 日期详情 | 只读、存在备注、存在加班 |
| S09 | 日期编辑 | 新增覆盖、修改、删除覆盖 |
| S10 | 统计 | 有数据、无数据、月份切换 |
| S11 | 提醒设置 | 权限允许、拒绝、系统关闭 |
| S12 | 主题设置 | 跟随系统、浅色、暗黑 |
| S13 | 桌面挂件设置 | 小/中/大、透明度、置顶、锁定 |
| S14 | 数据管理 | 导出、导入预览、冲突、清空确认 |
| S15 | 同步 | 未登录、同步中、成功、冲突、离线 |

## 优先实现页面

S02 → S03 → S05 → S06 → S08/S09 → S11 → S10 → S13。
